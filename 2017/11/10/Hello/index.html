百度百科解释：极简主义是一种生活以及艺术上的风格，本义为追求极致简约的呈现效果，并且不接受任何违反这一形态的任何事物。

极简主义涉及视觉艺术的不同领域，极简主义的核心原则基本都是优雅地保留关键性的、引起观者注意力的部分。将线条、形状、色彩、留白、构图等一切元素有效地组织起来。

今天我们在生活的各个领域都可以看得到极简主义的影子：建筑、艺术、摄影、文学、音乐、UI 设计，甚至食物。

极简风并不是简单的做减法，这也是极简设计最棘手的地方。想必设计师都明白“少即是多”这个概念，但是要怎么在实际设计中践行这种极简设计的理念呢？

极简设计的钟，貌似合理，其实是建立在人们已经有了对表盘认知的基础上的。如果是一个未学过数字表盘没有建立钟表意识的小孩，就认不出时间。

无刻度也不利于孩子学习认识时钟。所以，对于 UI 界面的一切简化都应是建立在用户已经形成认知的基础之上的。

极简主义是以满足用户需求为前提的，所以所谓极简主义就是撇除一切花里胡哨的东西，直达用户使用产品的目的。另外撇除花里胡哨的东西，同时还要保证美观，设计整体上就走向了一个让用户很难感知的方向。

移除额外的元素

在设计当中，每个元素都有其存在的意义。

在极简主义的设计当中，整个设计中每一个元素都应该是有目的、有针对性的存在，没有多余的、无用的元素。所以，极简主义的设计应该只保留绝对必要的那些组件。

简约配色

太多的色彩可能会对一个极简的设计产生负面的影响。

留白

充满呼吸感的布局是极简主义设计中常见的元素，也几乎是必要的组成部分。

留白指的是留出空间，它并不一定必须是白色。在极简风的设计当中，留白的存在让视觉焦点更为突出，此外，它还塑造出整个设计的空间感和区域，呈现整体框架和布局。
